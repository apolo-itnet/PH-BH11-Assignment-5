What are the different ways to select an element in the DOM?
    DOM-এ এলিমেন্ট সিলেক্ট করার উপায় :
    1 ID ব্যবহার করে : document.getElementById("id")
    2 Class ব্যবহার করে : document.getElementsByClassName("class")
    3 Tag ব্যবহার করে : document.getElementsByTagName("tag")
    4 Single Selector ব্যবহার করে : document.querySelector("selector")
    5 All Selector ব্যবহার করে : document.querySelectorAll("selector")
What is the difference between innerHTML, innerText, and textContent ?
    The Difference of innerHTML, innerText, and textContent
    # Property Explain HTML Process or not? Hidden Text read or not?
    1 innerHTML এলিমেন্টের ভেতরের HTML কোড (ট্যাগসহ) দেয় বা সেট করে। ✅ Yes ✅ Yes
    2 innerText শুধু দৃশ্যমান টেক্সট দেয়, ট্যাগ বা স্ক্রিপ্ট বাদ দিয়ে। ❌ No ❌ No

What is event delegation in the DOM?
Event Delegation : DOM (Document Object Model)-এ একটি Parent Element এ addEventListner যোগ করে Child Element এর Event কে ধরা। এতে প্রতিটি Child এর জন্য আলাদা addEventListner যুক্ত করা লাগে না।
What is event bubbling in the DOM?
Event Bubbling : DOM (Document Object Model)-এ DOM-এ event child element থেকে parent element এর দিকে ছড়িয়ে পড়ে।
How do you create, add, and remove elements using JavaScript?
    Create, Add, Remove Elements
    # Process Method Explain
    1 Create Element let div = document.createElement("div") এটি একটি নতুন HTML এলিমেন্ট (যেমন div) তৈরি করে, যা পরে কাস্টমাইজ করা যায়।
    2 Add Element parentElement.prepend(div) এটি প্যারেন্টের শুরুতে এলিমেন্ট যোগ করে।
    document.body.append(div) এটি তৈরি করা এলিমেন্টকে পেজের body-তে শেষে যোগ করে।
    3 Remove Element div.remove() এটি সরাসরি এলিমেন্টটিকে DOM থেকে মুছে ফেলে।
    parentElement.removeChild(div) প্যারেন্ট থেকে নির্দিষ্ট চাইল্ড এলিমেন্টটি সরায়।
